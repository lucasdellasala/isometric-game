# 🎮 Isometric Game

![isometric-game-02](https://github.com/user-attachments/assets/0b2c6ee0-ba1f-4b35-bd17-1f8e3edf2de7)

A learning project to master Rust by building an isometric RPG/exploration game from scratch — no game engine, just SDL2.

Inspired by the aesthetic of Fallout 1: pre-rendered 2D isometric graphics, tile-based maps, fog of war, and pathfinding.

## 🚀 Run it

```bash
cargo run
```

## 📋 Requirements

- [Rust](https://rustup.rs/)
- [CMake](https://cmake.org/download/) (SDL2 compiles automatically via the `bundled` feature)
